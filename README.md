# FKS_HanziPlay

Interactive Chinese character learning games for **Fornebu Kinesisk Skole** (福内布中文学校).

## 🎮 Games Included

| Game | Description |
|------|-------------|
| 🃏 Flashcards | Flip to reveal pinyin |
| 🎡 Wheel of Fortune | Spin and remove learned characters |
| 🎋 Bamboo Lottery | Shake the tube to draw a character |
| ✅ Pinyin Quiz | Choose the correct pinyin (4 options) |
| 📖 Character Select | Choose the correct character by pinyin |
| 🎯 Pairing Game | Match Chinese characters with pinyin |
| 🎴 Memory Match | Flip cards to match character and pinyin |

## 🔐 Password Protection

- Password required to access all games
- Login is remembered for 24 hours
- Right-click, Ctrl+S, F12, and Developer Tools are disabled

## 📖 Usage for Teachers

1. Visit the website (link provided by your school)
2. Enter the password
3. Select a book and lesson
4. Choose characters to practice
5. Select any game to start learning

## 🛡️ Security Features

- Password authentication (SHA256)
- Disabled right-click menu
- Disabled save shortcuts (Ctrl+S, Ctrl+U)
- Disabled Developer Tools (F12, Ctrl+Shift+I/J/C)
- Developer Tools detection (auto-refresh)
